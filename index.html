ecracker.net

A Virtual Christmas Cracker

Created by Terminal Velocity

What is it?

  • Two players "pull" a Christmas cracker using their phones
  • The phone with the highest acceleration wins
  • You then get a joke from:
    • /r/Jokes
    • /r/ProgrammingHumour

Tech Stack

  • Node.js
    • Express
    • Socket.io
  • MongoDB
  • Vue.js
  • Webpack
  • AWS Lightsail
